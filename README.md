# Code no Hero

**Code no Hero** is an anime-themed web development learning platform designed to help beginners unleash their coding potential, one line at a time. The website is crafted with plain HTML, CSS, and JavaScript, aimed at providing an interactive and fun learning experience. This project is built with the purpose of making web development accessible to a younger audience (18 years and below) who love anime.

## Features

- **Dark Mode**: Toggle between light and dark themes for a personalized browsing experience.
- **Anime-themed UI**: A vibrant and engaging user interface inspired by anime aesthetics.
- **CTA Button**: A fun call-to-action button that plays a Naruto jutsu sound when clicked.
- **Responsive Design**: The site is fully responsive, optimized for both desktop and mobile devices.
- **Interactive Lessons**: The site will soon include a collection of lessons to kickstart your web development journey.

## Technologies Used

- **HTML**: Structure of the web pages.
- **CSS**: Styling and animations for an engaging user interface.
- **JavaScript**: Interactive functionality, including dark mode toggle and sound effects.
- **Fonts**: [Orbitron Font](https://fonts.google.com/specimen/Orbitron) for a modern, futuristic look.

## Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/yourusername/code-no-hero.git
````

2. Navigate to the project folder:

   ```bash
   cd code-no-hero
   ```

3. Open `index.html` in your browser.

## Structure

```
/code-no-hero
  ├── /assets
  │    ├── /images
  │    ├── /sounds
  ├── /src
  │    ├── /css
  │    │    └── style.css
  │    ├── /js
  │    │    └── main.js
  ├── index.html
  ├── README.md
```

* `/assets/images`: Contains all the images (e.g., background, logos, etc.).
* `/assets/sounds`: Contains sound files like the Naruto jutsu sound effect.
* `/src/css`: Contains the styles for the website.
* `/src/js`: Contains JavaScript files for dark mode toggle and interaction.
