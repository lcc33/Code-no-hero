HTML Lessons

What does HTML stand for?

HyperText Markup Language

What is the purpose of the <head> tag?

It contains meta-information about the document, such as title and links to scripts and stylesheets.

How do you create a hyperlink in HTML?

Using the <a> tag: <a href="url">Link Text</a>

Which tag is used to display an image?

<img src="image.jpg" alt="Description">

How do you create an unordered list?

With <ul> for the list and <li> for each item.

What is the function of the <title> tag?

It sets the title of the web page shown in the browser tab.
